1. Run `npm init -y` to create package.json. Update it with:
   - `"name": "english-tutor-mcp"`

   - `"engines": { "node": ">=24.0.0" }`
   - Scripts:
     - `"dev": "tsx watch src/server.ts"`
     - `"build": "tsc"`
     - `"start": "node dist/server.js"`
     - `"db:generate": "drizzle-kit generate"`
     - `"db:migrate": "drizzle-kit migrate"`

2. Install production dependencies (exact versions pinned):
   ```
   npm install hono @hono/node-server drizzle-orm pg zod
   ```
   After install, verify zod version is 3.25.x in package-lock.json. If npm installed zod v4, force: `npm install zod@^3.25.0`

3. Install dev dependencies:
   ```
   npm install -D typescript @types/node @types/pg tsx drizzle-kit
   ```

4. Create `tsconfig.json`:
   ```json
   {
     "compilerOptions": {
       "target": "ES2022",
       "module": "NodeNext",
       "moduleResolution": "NodeNext",
       "outDir": "dist",
       "rootDir": "src",
       "strict": true,
       "esModuleInterop": true,
       "skipLibCheck": true,
       "forceConsistentCasingInFileNames": true,
       "resolveJsonModule": true,
       "declaration": true,
       "declarationMap": true,
       "sourceMap": true
     },
     "include": ["src/**/*"],
     "exclude": ["node_modules", "dist"]
   }
   ```

5. Create `.gitignore`:
   ```
   node_modules/
   dist/
   .env
   *.tsbuildinfo
   ```

6. Create `.env.example`:
   ```
   DATABASE_URL=postgresql://user:password@localhost:5432/english_tutor
   PORT=3000
   API_KEY=your-api-key-here
   ```

7. Create the `src/` directory: `mkdir -p src`

8. Create a minimal `src/index.ts` placeholder so `tsc` has something to compile:
   ```typescript
   // Entry point — replaced by server.ts in plan 01-03
   export {};
   ```

9. Run `npx tsc --noEmit` to verify TypeScript compiles with no errors.
